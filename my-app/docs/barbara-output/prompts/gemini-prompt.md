# Task VS-B1: Generate 8 Barbara Narrative Files

## Context
You are working on Visione Sostenibile, an Italian gardening company website (Next.js 16 + Convex + Tailwind v4).
The company is based in Torino, Piemonte, founded by Andrea Giordano. Biodinamico approach.

## Your Task
Read the storytelling document at: my-app/Visione Sostenibile_struttura storytelling.md

From this document, generate 8 structured narrative files. The document contains raw content for all these areas - your job is to EXPAND, STRUCTURE, and REFINE it into production-ready narrative architecture. Each file should be self-contained with clear sections.

## Output Files (create in docs/barbara-output/narrative/)

1. **positioning.md** — Brand positioning. Extract from sections "Posizionamento" and "Manifesto". Include: tagline, positioning statement, 3 key messages, B2C vs B2B differentiation, "scarto logico" anti-rhetoric hooks.

2. **metodo.md** — The sustainable method. Extract from "Metodo" section. Structure the 6-step process (Ascolto, Progetto, Piano lavori, Regia unica, Realizzazione, Manutenzione). Include deliverables, FAQ metodo, and "errori comuni" section.

3. **servizi.md** — Complete service architecture. Extract from "Servizi" and "Box Servizi Plus". Structure 3 levels (Verde sostenibile/Outdoor living/Acqua & benessere). Include Check-up Sostenibile offer, piano lavori infographic text, and Patto Visione Sostenibile.

4. **pacchetti-b2c.md** — B2C packages. Extract from "Pacchetti B2C". Structure 3 tiers (Base/Evoluto/Premium) with clear inclusions, ideal-for text, and CTAs.

5. **pacchetti-b2b.md** — B2B packages + TOV. Extract from "Pacchetti B2B" and "TOV_B2B". Structure 3 tiers (Base/Evoluto/Premium), B2B tone of voice, contractor positioning.

6. **faq.md** — Complete FAQ. Extract and merge all FAQ sections. Organize by category: General, Metodo, Servizi, Prezzi, Lombardia, Manutenzione. Minimum 12 Q&A pairs.

7. **case-study.md** — Case study templates. Extract from "Esempi casi studio" and "Box Servizi Plus" #3. Create 3 structured templates (Giardino Assetato, Riqualificazione Senza Sprechi, Terrazzo Esposto). Each with: Problema, Scelte, Risultato, Manutenzione.

8. **struttura-sito.md** — Complete site structure map. Extract from "Website Struttura" and "LP". Map all sections for each page type: Homepage (13 sections), Chi Siamo, Servizi, Metodo, Contatti, B2C, B2B. Include CTA placement and micro-copy for each section.

## Tone & Style Rules
- Italian language ONLY
- Anti-rhetoric: concrete, specific, no buzzwords
- Barbara methodology: every statement must have a "perche" (why)
- Andrea's voice: passionate but grounded, technical but human
- Differentiator: "contractor del verde" = un referente, squadra su misura
